# 🚀 Startup Validator

An AI-powered web application built with **React** that helps entrepreneurs, investors, and startup enthusiasts evaluate startup ideas before investing time and resources. The application analyzes different aspects of a startup idea and provides insights based on predefined validation criteria.

## 📖 Overview

Startup Validator is a modern web application developed using **React**. It provides an interactive platform where users can validate startup ideas based on multiple business parameters such as market demand, competition, feasibility, scalability, customer need, and innovation.

The project aims to help users make informed decisions by offering a structured evaluation process through a clean and responsive user interface.

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/startup-validator.git
```

Go to the project directory

```bash
cd startup-validator
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Open your browser

```
http://localhost:5173
```

---

## 🚀 Build for Production

```bash
npm run build
```

Preview production build

